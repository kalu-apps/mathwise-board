# Deploy на Amvera (whiteboard-only)

## Цель
Развернуть публичный demo-стенд коллективного урока:
- учитель входит по статичному логину/паролю;
- ученик подключается по временной invite-ссылке без авторизации;
- остальной функционал платформы скрыт (`whiteboard-only`).

## Базовые переменные окружения
Задать в настройках приложения:

- `VITE_SHOWCASE_MODE=realtime`
- `VITE_WHITEBOARD_ONLY=1`
- `VITE_WHITEBOARD_TEACHER_LOGIN=teacher@axiom.demo`
- `VITE_WHITEBOARD_TEACHER_PASSWORD=magic`

Опционально:
- `PORT` (если требуется платформой явно, иначе используется порт по умолчанию).

## Команды сборки и запуска
- Build: `npm ci && npm run build:showcase`
- Start: `npm run backend:start:showcase`

## Минимальный релиз-чеклист
1. Открыть `/`, авторизоваться как учитель.
2. Проверить авто-переход в активный коллективный урок.
3. Нажать «Скопировать ссылку приглашения».
4. Открыть `/workbook/invite/:token` в инкогнито, ввести имя, войти в сессию.
5. Проверить синхронизацию объектов доски между учителем и учеником.
6. Дождаться истечения TTL invite (30 минут) и проверить, что ссылка больше не пускает.

## Быстрое отключение публичного доступа
Варианты:
- остановить/заморозить приложение в Amvera;
- удалить приложение в Amvera;
- сменить пароль учителя (`VITE_WHITEBOARD_TEACHER_PASSWORD`) и перезапустить приложение.
