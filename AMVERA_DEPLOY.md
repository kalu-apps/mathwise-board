# Deploy на Amvera (Math Realtime Board Service)

## Цель
Развернуть отдельный микросервис доски:
- преподаватель входит по фиксированному аккаунту;
- ученик подключается по временной invite-ссылке;
- доступен только whiteboard-функционал.

## Переменные окружения
Задать в настройках приложения Amvera:

- `VITE_BOARD_MODE=realtime`
- `WHITEBOARD_ONLY=1`
- `VITE_WHITEBOARD_TEACHER_LOGIN=teacher@axiom.demo`
- `VITE_WHITEBOARD_TEACHER_PASSWORD=magic`

Опционально:
- `PORT`
- `VITE_BOARD_AUTO_LOGIN_EMAIL`
- `VITE_BOARD_AUTO_LOGIN_PASSWORD`

## Команды сборки и запуска
- Build: `npm ci && npm run build:board`
- Start: `npm run backend:start:board`

## Проверка после релиза
1. Открыть `/` и авторизоваться как преподаватель.
2. Убедиться, что открывается сессия доски.
3. Нажать «Скопировать ссылку приглашения».
4. Открыть `/workbook/invite/:token` в отдельном браузере, ввести имя и подключиться.
5. Проверить realtime-синхронизацию действий на доске.
6. Проверить, что после завершения урока invite-ссылка больше не пускает в сессию.

## Отключение публичного доступа
- остановить/заморозить приложение в Amvera;
- удалить приложение;
- сменить пароль преподавателя и перезапустить сервис.
