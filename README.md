# Math Realtime Lesson Demo

Демо-витрина сценария коллективного урока: учитель, участники, приглашения, синхронная работа на доске.

> Публичный showcase-репозиторий: часть внутренней инфраструктуры сознательно не публикуется. Правила использования и ограничения — в `SHOWCASE_NOTICE.md` и `LICENSE`.

## Что показано
- Старт/завершение коллективной сессии.
- Ролевая модель teacher/student.
- Приглашение участников в урок.
- Синхронизация состояния в процессе урока.
- Управление доступом участника к инструментам.

## Стек
- React + TypeScript + Vite
- MUI
- mock API/server + realtime события

## Быстрый старт
```bash
npm install
npm run dev:showcase
```

Откроется whiteboard-only сценарий:
- `/` — старт коллективного урока
- `/workbook/session/:sessionId` — доска урока
- `/workbook/invite/:token` — вход ученика по временной ссылке

## Демо-доступ
- Учитель: статичный логин/пароль из env (`VITE_WHITEBOARD_TEACHER_LOGIN` + `VITE_WHITEBOARD_DEMO_PASSWORD`).
- Ученик: без логина и пароля, по временной ссылке приглашения.

В showcase режиме включен `VITE_WHITEBOARD_ONLY=1`: прочие разделы платформы скрыты из роутинга.
Вход по magic-link отключен на mock API.

### Схема урока
1. Учитель авторизуется и автоматически попадает в активный коллективный урок (или создается новый).
2. Учитель копирует временную ссылку приглашения внутри урока и отправляет ученику.
3. Ученик открывает ссылку, вводит имя и сразу попадает в урок без авторизации.
4. Ссылка действует ограниченное время (по умолчанию 30 минут), затем становится неактивной.

## Что оценивать на собеседовании
- Реализация ролей и прав.
- Алгоритм realtime-синхронизации.
- Устойчивость сессии до/после завершения урока.
- UX для преподавателя при управлении участниками.

## Ограничения
- Видео отключено, акцент на доске и совместной работе.
